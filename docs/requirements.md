# 需求整理

## 目标

构建一个个人使用的本地资料问答和文字自动填表工具。系统从本地文档数据库中检索信息，生成可追溯回答，并能把检索到的文字结果填写进 Excel 表格。

## 功能范围

### 资料入库

- 支持 PDF、Excel、Word、TXT、Markdown。
- 提取文本并切分为 chunk。
- 每个 chunk 保存来源 metadata。

示例 metadata：

```json
{
  "file": "设备资料.xlsx",
  "path": "docs/设备资料.xlsx",
  "type": "xlsx",
  "sheet": "Sheet1",
  "row_start": 12,
  "row_end": 20
}
```

PDF 示例：

```json
{
  "file": "说明书.pdf",
  "path": "docs/说明书.pdf",
  "type": "pdf",
  "page": 5
}
```

### 手动查询

- 用户输入问题。
- 系统从向量库检索相关文本。
- 大模型只基于检索结果回答。
- 返回：
  - 答案
  - 原文摘录
  - 来源文件
  - PDF 页码
  - Excel Sheet 和行号范围

### 自动填表

- 上传待填写 Excel。
- 第一版约定：
  - 第一列是字段名。
  - 第二列是待填写值。
- 系统逐个字段查询资料库。
- 抽取文字值并写入第二列。
- 新增 `填写依据` Sheet。

填写依据字段：

```text
字段名
填写值
来源文件
位置
原文摘录
状态
```

状态：

```text
已填写
需要确认
未找到
```
